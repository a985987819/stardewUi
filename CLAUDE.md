## Agent skills

### Issue tracker

Issues for this repo are tracked in GitHub Issues. See `docs/agents/issue-tracker.md`.

### Triage labels

This repo uses the default triage label vocabulary (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses a single-context layout. Skills should look for a root `CONTEXT.md` and `docs/adr/`. See `docs/agents/domain.md`.
